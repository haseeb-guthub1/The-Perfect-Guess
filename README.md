# 🎯 Perfect Guess Game

The Perfect Guess Game is a beginner-friendly Python project that generates a random number and challenges the player to guess it. After each incorrect guess, the program provides hints indicating whether the guessed number is too high or too low until the correct number is found.

## 🚀 Features
- Generates a random number using Python's `random` module.
- Accepts user input for guesses.
- Provides hints after every incorrect guess.
- Counts the number of attempts taken.
- Displays a congratulatory message when the correct number is guessed.

## 🛠️ Technologies Used
- Python 3
- Random Module
